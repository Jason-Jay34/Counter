**Counter App**

A simple and interactive Counter Application built using HTML, CSS, and JavaScript. The app allows users to increase, decrease, and reset a number with a clean and responsive user interface.

**Features**

Increase the counter value

Decrease the counter value

Reset the counter to zero

Dynamic number updates using JavaScript

Simple and responsive design

**Technologies Used**

HTML – Structure of the application

CSS – Styling and layout

JavaScript – Application logic and interactivity

**How It Works**

The counter starts at 0.

Clicking the Increase button increments the value.

Clicking the Decrease button reduces the value.

Clicking the Reset button returns the counter to 0.

JavaScript listens for button clicks and updates the counter dynamically.
